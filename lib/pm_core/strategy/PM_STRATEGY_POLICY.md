# PM STRATEGY POLICY

## PURPOSE

Defines lawful PM strategy behavior inside AI_GO.

PM strategy is advisory interpretation.

PM strategy is not execution.

PM strategy is not mutation by default.

PM strategy does not create authority.

## REQUIRED FLOW

Any PM strategy path involving child-core context must preserve:

source or provider signal  
→ RESEARCH_CORE  
→ engine curation  
→ adapter  
→ PM interpretation  
→ PM strategy  
→ PM routing  
→ child-core ingress  

Provider retrieval is reserved for RESEARCH_CORE only.

Child-core strategy context remains curated, adapter-shaped, bounded, and advisory unless governed mutation is explicitly approved.

## PM STRATEGY MAY

- interpret governed context
- identify possible routing posture
- preserve lineage metadata
- preserve authority boundaries
- preserve execution_allowed = false
- support operator understanding

## PM STRATEGY MAY NOT

- execute actions
- mutate workflow directly
- mutate project truth directly
- write memory outside governance
- write outcome feedback outside governance
- create child-core authority
- create recommendation authority
- override governance

## MUTATION LAW

Persistence = Mutation.

If strategy evidence, strategy trace, receipt, memory record, outcome reference, dashboard packet, latest snapshot, or strategy artifact is persisted, it must be explicit, classified, traceable, and governed.

All mutation must pass:

State Authority  
→ Watcher  
→ Canon  
→ Request Governor  
→ Execution Gate  
→ Cross-Core Integrity  

## OUTCOME LAW

Outcome references are advisory only.

Outcome persistence is classified mutation.

Outcome references cannot alter workflow, routing, decision, or execution authority.

## EXECUTION LAW

execution_allowed = false by default.

PM strategy cannot trigger execution.

PM strategy only prepares bounded advisory posture.

## SUMMARY

PM strategy supports governed interpretation only.

It cannot mutate, execute, retrieve providers, or create authority.
