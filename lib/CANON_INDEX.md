# Canon Index

## Purpose

The Canon Index provides a structured listing of canonical governance documents stored in the `lib/` layer.

This index allows system authorities to locate foundational documents quickly without altering their origin or authority.

---

## Canonical Documents

### System Governance

- `_LIB_LAYER.md`
- `DOCUMENT_LIFECYCLE_POLICY.md`

### Registry Documents

- `DOCUMENT_REGISTRY.json`
- `LIFECYCLE_INDEX.json`

---

## Index Rules

1. Only canonical documents may appear in this index.
2. Each entry must correspond to a document registered in `DOCUMENT_REGISTRY.json`.
3. Deprecated or archived documents must be removed from the active canon list but remain traceable through lifecycle records.
4. Canonical documents must preserve the originating authority.

---

## Summary

The Canon Index is the primary navigation reference for canonical system governance documents within the `lib/` layer.
